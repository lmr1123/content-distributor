# 安装和使用指南

## 快速开始

### 第一步：安装扩展

1. 打开 Chrome 浏览器
2. 在地址栏输入：`chrome://extensions/`
3. 在右上角找到并开启「开发者模式」开关
4. 点击左上角的「加载已解压的扩展程序」按钮
5. 选择 `content-distributor` 文件夹
6. 看到"全平台内容分发助手"扩展卡片即安装成功

### 第二步：获取微信文章

1. 找一篇你想分发的微信公众号文章
2. 复制文章链接（格式如：https://mp.weixin.qq.com/s/xxxxx）
3. 点击浏览器工具栏中的扩展图标（紫色分发图标）

### 第三步：导入和编辑

1. 在左侧"导入文章"区域粘贴链接
2. 点击"获取内容"按钮
3. 等待几秒钟，内容会自动填充到编辑器
4. 在右侧编辑器中修改或完善内容
5. 可以在左侧修改标题

### 第四步：选择平台并分发

1. 在底部"选择发布平台"区域勾选目标平台
   - 小红书（限1000字）
   - 知乎（无限制）
   - 简书（无限制）
   - 今日头条（推荐5000字内）
   - 哔哩哔哩（专栏）
2. 点击"开始同步分发"按钮
3. 系统会依次打开所选平台的发布页面
4. 内容会自动填充，检查无误后即可发布

## 使用技巧

### 内容优化

- 点击编辑器上方的"格式优化"按钮可以自动优化段落间距
- 小红书建议使用更口语化的表达
- 知乎适合深度长文
- 今日头条推荐吸引人的标题

### 封面图片

- 点击"更换封面"可以上传自定义封面图
- 不同平台对封面尺寸要求不同，建议使用 16:9 或 3:4 比例

### 自动保存

- 编辑的内容会自动保存到浏览器本地
- 下次打开时会自动恢复上次的内容
- 点击"清空内容"可以重新开始

## 故障排除

### 问题1：无法获取微信文章

**原因**：
- 链接格式不正确
- 文章需要登录才能查看
- 网络问题

**解决方法**：
1. 确认链接包含 `mp.weixin.qq.com`
2. 确保文章是公开的
3. 检查网络连接

### 问题2：自动填充不生效

**原因**：
- 平台页面未完全加载
- 平台页面结构发生变化
- 未登录目标平台

**解决方法**：
1. 等待几秒后刷新页面
2. 先登录目标平台账号
3. 手动复制粘贴内容

### 问题3：图片无法显示

**原因**：
- 微信图片有防盗链保护

**解决方法**：
1. 手动下载图片
2. 上传到目标平台

## 高级功能

### 自定义平台

你可以在 `js/content-scripts/` 目录添加新平台的支持：

1. 创建新的 JS 文件，如 `weibo.js`
2. 参考现有文件编写自动填充逻辑
3. 在 `manifest.json` 中注册新的 content script

### 调试模式

打开浏览器控制台（F12）可以查看详细的运行日志。
