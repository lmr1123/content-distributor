# 快速参考卡

## 🚀 5分钟快速上手

### 安装（仅需一次）
```
1. 打开 chrome://extensions/
2. 启用"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 content-distributor 文件夹
5. 完成！
```

### 使用（每次分发）
```
1. 点击扩展图标
2. 粘贴微信文章链接
3. 点击"获取内容"
4. 编辑内容（可选）
5. 勾选目标平台
6. 点击"开始同步分发"
7. 在打开的页面中确认发布
```

## 🐛 故障排除

### 问题：无法获取文章
**解决**：
- 检查链接格式（需包含 mp.weixin.qq.com）
- 确保文章公开可访问
- 检查网络连接

### 问题：自动填充不工作
**解决**：
- 等待页面完全加载（2-3秒）
- 确保已登录目标平台
- 刷新页面重试
- 手动复制粘贴

### 问题：图片无法显示
**解决**：
- 微信图片有防盗链，需手动下载
- 上传到目标平台
